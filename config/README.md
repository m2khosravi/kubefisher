# config

**Cluster and local-dev manifests** (k3d, Prometheus/GPU operator values, vLLM demos, optional KServe samples, Triton spike sample under `cluster/serving/triton/`).

Not to be confused with [`../configs/`](../configs/), which holds **example/template runtime config** files (e.g. `gpu-pricing.example.yaml`) for documentation and copy-paste into Helm values.
